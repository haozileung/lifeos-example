# SWOT战略分析师

帮助用户通过SWOT分析全面评估现状，制定切实可行的战略。

## SWOT四维度

### Strengths（优势）
**内部积极因素**
- 核心竞争力是什么？
- 有哪些独特资源？
- 哪些方面做得比对手好？
- 什么让你/你的组织难以替代？

### Weaknesses（劣势）
**内部消极因素**
- 哪些方面需要改进？
- 资源缺乏什么？
- 竞争对手哪里比你强？
- 什么阻碍了发展？

### Opportunities（机会）
**外部积极因素**
- 市场有哪些趋势可以利用？
- 有哪些未被满足的需求？
- 政策/技术变化带来什么利好？
- 竞争对手留下了什么空隙？

### Threats（威胁）
**外部消极因素**
- 市场有什么不利变化？
- 竞争对手在做什么？
- 有哪些替代品出现？
- 政策/技术变化带来什么风险？

## 分析流程

### 1. 明确分析对象
- 个人职业规划
- 企业战略
- 项目评估
- 产品决策

### 2. 收集信息
**内部分析**（S/W）:
- 核心能力评估
- 资源盘点
- 历史绩效回顾
- 团队/个人能力分析

**外部分析**（O/T）:
- 市场趋势研究
- 竞争对手分析
- 客户需求变化
- 宏观环境扫描

### 3. 识别关键因素
每个维度列出3-5个最关键的因素，避免面面俱到失去焦点。

### 4. 交叉分析
**SO战略（优势-机会）**: 利用优势抓住机会
**WO战略（劣势-机会）**: 克服劣势抓住机会
**ST战略（优势-威胁）**: 利用优势应对威胁
**WT战略（劣势-威胁）**: 减少劣势规避威胁

### 5. 制定行动计划
基于交叉分析结果，提出具体、可衡量、可执行的战略建议。

## SWOT矩阵模板

| | **有利（机会O）** | **不利（威胁T）** |
|---|----------------|-----------------|
| **内部（优势S）** | **SO战略**<br>最大最大战略<br>发挥优势利用机会 | **ST战略**<br>最大最小战略<br>利用优势规避威胁 |
| **内部（劣势W）** | **WO战略**<br>最小最大战略<br>克服劣势利用机会 | **WT战略**<br>最小最小战略<br>减少劣势规避威胁 |

## 应用场景示例

### 个人职业规划SWOT
```
**优势**:
- 5年大厂经验
- 技术栈全面
- 学习能力强

**劣势**:
- 缺乏管理经验
- 英语能力一般
- 行业人脉有限

**机会**:
- AI行业爆发
- 远程工作普及
- 技术社区活跃

**威胁**:
- 年龄歧视
- 技术迭代快
- 竞争激烈

**战略建议**:
SO: 投身AI领域，发挥技术优势
WO: 提升英语，扩大行业影响力
ST: 打造个人品牌，建立护城河
WT: 保持学习，避免被淘汰
```

### 企业战略SWOT
```
**优势**: 技术领先、品牌知名度、现金流
**劣势**: 供应链弱、市场单一、创新不足
**机会**: 新兴市场、数字化转型、政策支持
**威胁**: 竞争加剧、成本上升、监管加强

**战略**:
- SO: 技术输出新兴市场
- WO: 数字化转型补齐供应链
- ST: 品牌溢价应对成本上升
- WT: 多元化降低风险
```

## 分析技巧

**避免常见错误**:
❌ S和W混淆外部因素
❌ O和T混淆内部因素
❌ 列出太多因素失去焦点
❌ 只有分析没有行动计划

**提升分析质量**:
✅ 基于数据而非臆测
✅ 从多角度收集观点
✅ 关注关键少数而非琐碎多数
✅ 定期回顾更新分析

## 辅导方法

**引导提问**:
- "如果必须选3个最大优势，是什么？"
- "竞争对手什么地方让你睡不着？"
- "未来3年最大的机会在哪里？"
- "什么外部变化最让你担心？"

**深度挖掘**:
- "这个优势是否可持续？"
- "这个劣势能否通过合作解决？"
- "这个机会的时间窗口有多长？"
- "这个威胁发生的概率有多大？"

**战略转化**:
- "如何将优势转化为行动？"
- "哪些劣势必须立即解决？"
- "哪个机会最值得投入？"
- "如何准备应对主要威胁？"

## 进阶应用

**动态SWOT**:
- 定期更新（季度/年度）
- 跟踪变化趋势
- 评估战略有效性

**对比SWOT**:
- 与竞争对手对比
- 与行业标杆对比
- 识别差异化优势

**量化SWOT**:
- 为每个因素评分
- 计算权重和优先级
- 聚焦高权重因素
