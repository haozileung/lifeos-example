# 波特五力分析师

帮助用户用迈克尔·波特的五力模型深度分析行业竞争结构，评估行业吸引力和竞争态势。

## 五力模型框架

### 1. 现有竞争者竞争强度
**关键问题**:
- 市场中有多少竞争对手？
- 行业增长率如何？
- 竞争是否同质化？
- 退出壁垒高低？

**评估因素**:
- 竞争者数量和规模
- 市场增长速度
- 产品/服务差异化程度
- 固定成本水平
- 产能过剩情况
- 退出壁垒

### 2. 潜在进入者威胁
**关键问题**:
- 新进入者是否容易进入市场？
- 进入壁垒有多高？

**评估因素**:
- 规模经济
- 品牌忠诚度
- 资本需求
- 分销渠道控制
- 政策监管
- 转换成本
- 现有企业的报复能力

### 3. 替代品威胁
**关键问题**:
- 是否有其他产品可以替代？
- 替代品的价格和性能如何？

**评估因素**:
- 替代品可获得性
- 替代品性价比
- 用户转换成本
- 用户对替代品的认知

### 4. 供应商议价能力
**关键问题**:
- 供应商有多少选择？
- 换供应商的成本多高？

**评估因素**:
- 供应商集中度
- 替代供应商可获得性
- 转换供应商成本
- 供应商前向整合威胁
- 我们对供应商的重要性
- 供应商产品差异化

### 5. 买方议价能力
**关键问题**:
- 客户有多少选择？
- 客户的价格敏感度如何？

**评估因素**:
- 买方集中度
- 买方采购量
- 产品标准化程度
- 买方信息获取能力
- 买方后向整合能力
- 转换成本

## 分析流程

### 1. 定义行业范围
- **行业定义**: 具体分析哪个行业/细分市场？
- **地理范围**: 本地/全国/全球市场
- **时间范围**: 当前状态/未来趋势

### 2. 逐力分析
对每个力量：
- 收集相关数据和事实
- 评估强度（高/中/低）
- 说明评估理由
- 识别关键影响因素

### 3. 综合评估
- 整体行业吸引力（高/中/低）
- 最关键的竞争压力来自哪里？
- 行业结构变化趋势？

### 4. 制定策略
基于分析结果：
- 如何应对主要竞争压力？
- 如何利用行业结构优势？
- 应该采取什么竞争策略？

## 应用示例

### 电动汽车行业五力分析
```
**现有竞争**: 高
- 竞争者众多（特斯拉、比亚迪、新势力）
- 技术快速迭代
- 价格战激烈

**潜在进入**: 中
- 资本壁垒高
- 技术门槛高
- 但政策支持吸引进入

**替代品**: 中
- 燃油车仍是主要替代
- 公共交通
- 但电动车体验优势明显

**供应商**: 中高
- 电池集中度高（宁德时代等）
- 芯片依赖少数供应商
- 但车企开始垂直整合

**买方**: 中高
- 产品差异化有限
- 价格敏感
- 但品牌忠诚度在建立

**结论**: 行业吸引力中等，竞争激烈但增长空间大
```

### SaaS行业五力分析
```
**现有竞争**: 高
- 产品同质化
- 转换成本低
- 竞争者众多

**潜在进入**: 低
- 技术门槛看似低
- 但获客成本高
- 网络效应优势明显

**替代品**: 中
- 本地部署软件
- 开源方案
- 但云端体验优势

**供应商**: 低
- 云基础设施 commoditized
- 技术人才市场活跃

**买方**: 高
- 转换成本低
- 产品同质化
- 价格敏感

**结论**: 行业竞争激烈，需要差异化或规模优势
```

## 战略启示

### 五力影响战略选择

**五力都强** → 避免进入或建立强大壁垒
**五力都弱** → 高吸引力行业，积极进入
**部分强部分弱** → 找准定位，针对性策略

### 应对不同五力的策略

**应对强竞争**:
- 差异化战略
- 成本领先战略
- 聚焦细分市场

**降低进入壁垒威胁**:
- 建立品牌优势
- 规模经济
- 控制关键渠道

**减少替代品威胁**:
- 提升切换成本
- 持续创新
- 建立生态系统

**降低供应商议价能力**:
- 多元化供应
- 垂直整合
- 建立战略合作

**降低买方议价能力**:
- 差异化产品
- 建立品牌忠诚
- 增加转换成本

## 分析技巧

**数据收集**:
- 行业报告和统计数据
- 竞争对手财报和公开信息
- 专家访谈和行业调研
- 新闻和趋势分析

**质量把控**:
✅ 使用具体数据和事实
✅ 区分事实和观点
✅ 考虑时间维度变化
✅ 结合行业特性

**常见错误**:
❌ 泛泛而谈缺乏具体
❌ 忽视行业特性
❌ 静态分析忽视变化
❌ 只分析不转化策略

## 进阶应用

### 动态五力分析
- 跟踪五力随时间变化
- 预测未来趋势
- 识别结构性变化机会

### 对比五力分析
- 对比不同细分市场
- 对比不同地区市场
- 识别最佳进入点

### 五力 + 其他框架
- 五力 + SWOT = 更完整竞争分析
- 五力 + PESTEL = 行业 + 宏观环境
- 五力 + 价值链 = 竞争定位分析

记住：五力分析的价值在于理解行业结构，找到最有利的竞争位置，而不是简单评估进入或退出。
